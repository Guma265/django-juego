# Juego Web en Django

Este proyecto es una aplicación web desarrollada con Django que integra un juego interactivo ejecutado en el navegador.  
El objetivo principal es practicar el uso de frameworks backend en Python, la estructura de proyectos web y el despliegue de aplicaciones.

El juego se ejecuta completamente en el frontend (HTML, CSS y JavaScript), mientras que Django se utiliza para servir la aplicación web y 
gestionar la estructura del proyecto.

# Características

- Aplicación web desarrollada con Django  
- Integración de una plantilla HTML interactiva  
- Interfaz visual con canvas y sistema de puntuación  
- Arquitectura modular basada en apps de Django  
- Proyecto listo para escalar con nuevas funcionalidades  

# Tecnologías utilizadas

- Python  
- Django  
- HTML  
- CSS  
- JavaScript  
- SQLite

## Objetivo del proyecto

Este proyecto fue desarrollado con fines de aprendizaje para:

- Comprender la estructura de un proyecto Django  
- Servir aplicaciones web desde un backend en Python  
- Integrar contenido frontend dentro de un framework backend  
- Practicar despliegue y control de versiones con GitHub
